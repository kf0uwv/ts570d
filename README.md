# Kenwood TS-570D Control Program

This is a serial control program for the Kenwood TS-570D radio. It allows for remote control of the radio's settings and functions through a serial connection, as well as diagnostics for serial connectinos. 

## Language

The program is written in Rust and uses the serial crate.

## Protocol

You can find the protocol in the manual on pages 70-81 https://www.kenwood.com/usa/Support/pdf/TS-570-English.pdf.
