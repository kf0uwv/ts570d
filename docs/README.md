# docs/

## TS-570D Instruction Manual

The Kenwood TS-570D instruction manual PDF is not committed to this repository (binary, ~4 MB).

**Download it before running the kenwood agent:**

```
curl -L -o docs/TS-570-English.pdf \
  https://www.kenwood.com/usa/Support/pdf/TS-570-English.pdf
```

The CAT (Computer Aided Transceiver) command reference is in booklet pages 70–81
(approximately PDF pages 76–87). This section defines all command codes, parameter
formats, response layouts, and valid ranges for the TS-570D.
